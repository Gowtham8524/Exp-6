# Exp - 6 STUDENT ONLINE EXAM REGISTRATION SYSTEM

# AIM:
To draw an UML diagram for student online exam registration system
# SRS (Procedure):
Procedure (SRS):

Create a login page for students and admin.

Allow students to register or log in using valid credentials.

Display list of available exams on the student dashboard.

Provide a form for students to fill in exam registration details.

Store the registration data in the database.

Allow admin to log in and view all student registrations.

Admin verifies and approves or rejects registrations.

Generate hall ticket or confirmation message for approved students.

Allow students to download or print their hall ticket.

Ensure security, accuracy, and proper record maintenance.
